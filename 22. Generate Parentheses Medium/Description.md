Given `n` pairs of parentheses, write a function to generate all combinations of well-formed parentheses.
 

#### Example 1:

<pre>
<code>
<b>Input:</b> n = 3
<b>Output:</b> ["((()))","(()())","(())()","()(())","()()()"]
</code>
</pre>

#### Example 2:

<pre>
<code>
<b>Input:</b> n = 1
<b>Output:</b> ["()"]
</code>
</pre>
 

#### Constraints:

- `1 <= n <= 8`
