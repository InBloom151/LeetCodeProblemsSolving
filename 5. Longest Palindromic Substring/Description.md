Given a string `s`, return the ***longest palindromic substring*** in `s`.

 

#### Example 1:

<pre>
<code>
<b>Input:</b> s = "babad"
<b>Output:</b> "bab"
<b>Explanation:</b> "aba" is also a valid answer.
</code>
</pre>

#### Example 2:

<pre>
<code>
<b>Input:</b> s = "cbbd"
<b>Output:</b> "bb"
</code>
</pre>
 

#### Constraints:

- `1 <= s.length <= 1000`
- `s` consist of only digits and English letters.
