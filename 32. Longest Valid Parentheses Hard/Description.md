Given a string containing just the characters `'('` and `')'`, return the length of the longest valid (well-formed) parentheses substring.
 

#### Example 1:

<pre>
<code>
<b>Input:</b> s = "(()"
<b>Output:</b> 2
</code>
</pre>

#### Example 2:

<pre>
<code>
<b>Input:</b> s = ")()())"
<b>Output:</b> 4
</code>
</pre>

#### Example 3:

<pre>
<code>
<b>Input:</b> s = ""
<b>Output:</b> 0
</code>
</pre>
 

#### Constraints:

- `0 <= s.length <= 3 * 104`
- `s[i] is '(', or ')'`
